    videomaps.earth

    This project lets you overlay videos of satellite imagery onto maps. You choose a place, set a timeframe, and the tool gets the data from a STAC endpoint. The result is a zoomable video on the map with very fine detail, down to changes that happen week by week.

    You'll sometimes see cloudy frames in the video. Don't worry, that's just part of having such detailed time steps.

    You can visit a few preset interesting places with this tool. Or, if you have access to a Python environment, go ahead and try this on any place in the world with any time steps.
